## Clone repository:
```bash
git clone https://github.com/PhanHoangKhang/music-site.git
```
## Run Front-end
```bash
cd client
npm install
npm run dev
```
## Run Back-end
```bash
cd server
npm install
npm run dev
```
# Environment variables
Create a .env file in the server directory and add:
```bash
PORT = 4000
MONGO_URI = 
```
